# Marcador

### En esta sección aprenderás cómo agregar, editar y borrar marcadores en el mapa.

1. Para interactuar con un marcador, deberemos acceder primeramente a los ajustes del mapa, este apartado se encuentra en la barra de herramientas.

2. Nos dirigiremos a los ajustes de marcador.

![Barr](/docs/images/menu_map.png)


Se abrirá el panel de ajustes de marcadores, tendremos tres opciones, agregar, editar y borrar.

![Menu Mapa](/docs/images/menu_marker.png)


## Agregar Marcadores

Para agregar un nuevo marcador, basta con hacer click sobre el botón.
Se habilitará la opción de agregar y tan sólo deberemos ir a la zona del mapa donde queremos insertarlo.


![Agregar Marcadores](/docs/images/add_marker.gif)
